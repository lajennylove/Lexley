Most Easy and Reliable Way to Connect Your WordPress Form Plugins to Trello & Slack .

== Description ==

Forms to Slack & Trello Integration.  is a WordPress  Integration Plugin its Create Trello Card or Slack Comment After a Wordpress form  Data Is Saved.
 It uses Trello And Sack API  for sending the Data. supported  Form Plugins are .


 * **[Contact form 7](https://wordpress.org/plugins/contact-form-7/) form Submission  to Slack & Trello Integration**
    * Text fields
    * Text Area fields 
    * Number           
    * tel              

 * **[ninja form](https://wordpress.org/plugins/ninja-forms/) Submission  to Slack & Trello Integration**
    * Text fields
    * Text Area fields 
    * Number           
    * tel              

 * **[Formidable](https://wordpress.org/plugins/formidable/) form  Submission  to Slack & Trello Integration**
    * Text fields
    * Text Area fields 
    * Number           
    * tel              

 * **[WPform](https://wordpress.org/plugins/wpforms-lite/)  Submission  to Slack & Trello Integration**
    * Text fields
    * Text Area fields 
    * Number           
    * tel              

 * **[weForms](https://wordpress.org/plugins/weforms/)  Submission  to Slack & Trello Integration**
    * Text fields
    * Text Area fields 
    * Number           
    * tel              

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `slatre` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
